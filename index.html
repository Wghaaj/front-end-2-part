Commands used in terminal:

  1. git checkout -b ... -- switching branches and creating new branches
  2. git --version -- helps you track different versions of your code
  3. cd.. -- returning the path to a local folder on your computer's disk
  4. ls -- lists the current directory contents
  5. git clone -- point to an existing repo and make a clone or copy of that repo at in a new directory, at another location
  6. git log -- displays committed snapshots
  7. git status -- displays the state of the working directory and the staging area
  8. git add -- adds a change in the working directory to the staging area
  9. git commit -m"..." -- take all of the changes that have been made locally and push them up to a remote repository.
  10. git push origin main -- push the current branch to the branch of the matching name in the remote repository
  11. git push origin HEAD -- push the current branch to the remote counterpart on github or some other server, if it exists
